自适应布局-rem

发表于 2018-03-23

由于移动端特殊性，本文讲的是如何使用rem实现自适应，或叫rem响应式布局，通过使用一个脚本就可以rem自适应，不用再为各种设备宽度不同而烦恼如何实现自适应的问题。

rem是相对于根元素html变化而变化的，我们只需要在根元素确定一个px字号，就可以算出元素的宽高。

原理

1.判断设备，对根元素html进行font-size的设置;

2.同时动态生成viewport，并设置initial-scale(缩放)的值。

使用方法

1.在页面引入这段js代码，同时页面不需要设置默认的viewport值;

2.根据设计稿上元素尺寸字号去写样式,例如某张图片的宽为200px，高为200px，那设置的宽应为(200/(设计稿宽度/10))rem,高为(200/(设计稿宽度/10))rem。

注意点

字体建议还是用px，使用rem容易出现带小数点的字体或者奇数字体的情况;

有些情况会出现设置字体大小无效，解决方法是设置display:inline-block。
